# Banking Application


## Run Locally :
Clone the Project
```
https://github.com/kundan799/Kelp_banking_application
``` 
Go to Client directory
```
cd Kelp_banking_Application
```
Install Dependencies
```
npm install
```

# Sample

node index.js CREATE ACC001 KELP
